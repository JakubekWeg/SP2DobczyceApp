# SP2DobczyceApp
Aplikacja stworzona dla Szkoły podstawowej nr 2 w Dobczycach oraz dla Gimnazjum w Dobczycach im. Jana Pawła II.

Zazwalam na korzystanie z kodu bez ograniczeń, lecz chcę aby w edycjach tego programu znalazło się nazwisko pierwotnego autora.
